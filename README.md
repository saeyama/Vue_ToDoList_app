# ToDoリストのアプリ

Vue.js 2系(CDN)  
データ保存先はLocal Strage  

**アプリ概要**  
メモの登録・更新・削除が出来るシンプルなアプリです。  
ローカル環境で使用可能です。

**使用方法**    
`clone`もしくは`fork`でローカルに落とし、index.htmlファイルをブラウザで開くと使用可能です！
